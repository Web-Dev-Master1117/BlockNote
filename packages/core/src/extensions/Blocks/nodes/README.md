# Node structure

We use the following document structure:

```xml
<blockgroup>
    <block>
        <content>Parent element 1</content>
        <blockgroup>
            <block>
                <content>Nested / child / indented item</content>
            </block>
        </blockgroup>
    </block>
    <block>
        <content>Parent element 2</content>
        <blockgroup>
            <block>...</block>
            <block>...</block>
        </blockgroup>
    </block>
    <block>
        <content>Element 3 without children</content>
    </block>
</blockgroup>
```
